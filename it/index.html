Benvenuti

This translation is a community translation, it has limited support and might not be up to date with the latest version in English. Questa traduzione è una traduzione community, ha un supporto limitato e potrebbe non essere aggiornata con l'ultima versione in inglese.

Volete far parte del movimento del software verde?

Se volete rendere le vostre applicazioni software più sostenibili e costruire una carriera nel settore del software verde, questo corso di formazione fa per voi. Vi insegnerà a costruire, mantenere ed eseguire applicazioni più ecologiche, indipendentemente dal dominio applicativo, dal settore, dalle dimensioni o dal tipo di organizzazione, dal linguaggio di programmazione o dal framework.

Se si supera l'esame associato, viene rilasciato un certificato di completamento dalla Linux Foundation. Alcuni datori di lavoro hanno creato dei moduli per facilitare la gestione dei crediti per il completamento di questo corso. Se il vostro datore di lavoro è in questo elenco, fate clic sul nome della vostra organizzazione per registrare i crediti. Se la vostra organizzazione desidera aggiungere il proprio modulo all'elenco, contattate il nostro team all'indirizzo help@greensoftware.foundation.

Che cos'è un professionista del software?​

Un professionista del software è chiunque sia coinvolto nella pratica di costruire, distribuire o gestire un'applicazione software. Questo include, ma non si limita a, le seguenti figure:

  • Sviluppatori
  • Professionisti Dev Ops
  • Testers
  • Architetti
  • SRE (Site Reliability Engineers)
  • Product Manager
  • UX Designer

Come ottenere il meglio da questo corso?​

  • Avere almeno un anno di esperienza nel settore della creazione, distribuzione o gestione di applicazioni software.
  • Completare ogni sezione in ordine dall'inizio alla fine.
  • Completare i check delle competenze alla fine di ogni sezione.
  • Se si desidera ottenere un certificato di completamento, è necessario sostenere l'esame tramite il sito web della Linux Foundation.

Al termine di questa formazione, comprenderete quanto segue:​

  • I principali fattori che determinano le emissioni di carbonio delle applicazioni software.
  • Le 3 azioni che potete intraprendere per ridurre le emissioni di carbonio del vostro software.
  • Come misurare le emissioni del software utilizzando il protocollo Greenhouse Gas (GHG) o la specifica Software Carbon Intensity (SCI).
  • I diversi tipi di obiettivi climatici aziendali, in modo da poter allineare le vostre proposte di software verde per contribuire a soddisfare gli impegni pubblici della vostra organizzazione.

Chi ha creato questa formazione?​

Questa formazione è stata creata dal team Principles & Patterns del Community Working Group della Green Software Foundation.

La formazione è open source e creative commons, il sorgente di questo sito può essere trovato sul nostro GitHub https://github.com/Green-Software-Foundation/training.
